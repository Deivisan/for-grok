# 02 - OPERAÇÕES

## ⚙️ Como o Atendimento Funciona na Prática

Fluxo completo desde o primeiro contato até o pós-venda.

---

## 📞 Fluxo de Atendimento

### Etapa 1: Captação do Cliente

**Canais de Entrada:**
1. **Site (analise.html)** - Cliente faz diagnóstico online
2. **WhatsApp Direto** - Indicação ou busca orgânica
3. **Redes Sociais** - Instagram, Facebook (mínimo atualmente)
4. **Indicações** - Boca a boca (principal hoje)

**Primeiro Contato:**
```
Cliente: "Oi, vi seu site. Meu PC tá muito lento."
Técnico: "Olá! Vou te ajudar. Pode me dizer:
1. Qual sistema operacional usa agora?
2. PC é desktop ou notebook?
3. Já fez o diagnóstico no nosso site?"
```

**Diagnóstico Prévio:**
- Cliente acessa https://deivisan.github.io/DeiviTech-Formatacao/analise.html
- Sistema detecta hardware automaticamente (CPU, RAM, GPU, SO)
- Ou cliente preenche formulário manual
- Técnico recebe especificações e avalia

---

### Etapa 2: Orçamento

**Análise do Problema:**
- Técnico avalia relatório de diagnóstico
- Identifica causa da lentidão (vírus, HD ruim, pouca RAM, etc)
- Sugere solução adequada

**Exemplo de Conversa:**
```
Técnico: "Analisando seu PC:
- Windows 7 desatualizado (fim do suporte)
- HD mecânico lento
- 4GB RAM (pouco para hoje)

Solução recomendada:
✓ Formatação Windows 10: R$ 25
✓ Upgrade SSD 240GB: R$ 120
✓ Upgrade RAM para 8GB: R$ 60
✓ Instalação: R$ 30

Total: R$ 235
Seu PC vai ficar 3x mais rápido."
```

**Negociação:**
- Cliente pode optar por parte do serviço
- Técnico sugere prioridades (SSD é mais impactante que RAM)
- Flexibilidade de pagamento (PIX, dinheiro, transferência)

---

### Etapa 3: Agendamento

**Opções de Atendimento:**

**A) Domiciliar (preferido pelo cliente)**
- Técnico vai até a casa/empresa do cliente
- Vantagem: Conveniência para cliente
- Desvantagem: Tempo de deslocamento
- Custo: Incluso no preço (dentro de Feira de Santana)

**B) Balcão (preferido pelo técnico)**
- Cliente leva o PC até o técnico
- Vantagem: Mais rápido, ambiente controlado
- Desvantagem: Inconveniência para cliente
- Local: Residência do técnico (home office)

**Agendamento Online:**
- Cliente acessa https://deivisan.github.io/DeiviTech-Formatacao/agendamento.html
- Seleciona serviço desejado
- Escolhe data e horário no calendário
- Preenche dados (nome, endereço, telefone)
- Recebe confirmação automática no WhatsApp

**Confirmação:**
```
WhatsApp automático:
"✅ Agendamento confirmado!
📅 Data: 15/02/2026
🕐 Horário: 14:00
🔧 Serviço: Formatação + SSD
📍 Local: Rua X, Nº Y

DeiviTech Formatação
(75) 98123-1019"
```

---

### Etapa 4: Execução do Serviço

**Checklist de Atendimento:**

**Antes de começar:**
- [ ] Verificar equipamento (liga? tela funciona?)
- [ ] Backup de dados (se contratado)
- [ ] Foto do estado atual (proteção contra contestação)
- [ ] Confirmar serviços acordados
- [ ] Receber pagamento (50% adiantado para serviços grandes)

**Durante o serviço:**
- [ ] Formatação do sistema
- [ ] Instalação de drivers
- [ ] Instalação de programas essenciais
- [ ] Configurações de otimização
- [ ] Testes de funcionamento
- [ ] Instalação de hardware (se aplicável)

**Programas Instalados (padrão):**
- Navegador: Google Chrome
- Player: VLC Media Player
- Compactador: WinRAR/7-Zip
- PDF: Adobe Reader/Sumatra
- Office: LibreOffice (gratuito) ou instalação do Office do cliente
- Antivírus: Windows Defender (já incluso no Win 10/11)
- Outros: Segundo necessidade do cliente

**Tempo Estimado:**
- Formatação simples: 1-2 horas
- Formatação + programas: 2-3 horas
- Com upgrade SSD: 3-4 horas
- Com upgrade SSD + RAM: 4-5 horas

---

### Etapa 5: Entrega

**Checklist Final:**
- [ ] Sistema iniciando corretamente
- [ ] Drivers funcionando (som, rede, vídeo)
- [ ] Programas abrindo
- [ ] Internet conectando
- [ ] Teste de velocidade (se SSD instalado)
- [ ] Mostrar para cliente o resultado
- [ ] Explicar o que foi feito
- [ ] Dicas de uso (manutenção preventiva)

**Dicas Entregues ao Cliente:**
```
"Seu PC está pronto! Algumas dicas:
1. Desligue corretamente (não force o botão)
2. Evite instalar programas desnecessários
3. Mantenha o Windows atualizado
4. Use o Chrome que instalei (é mais leve)
5. Se ficar lento de novo, me chama (garantia 30 dias)"
```

**Pagamento:**
- Receber restante do pagamento
- PIX preferido (rápido e sem taxa)
- Dinheiro (aceito)
- Transferência bancária
- **Não aceita cartão** (ainda não tem maquininha)

---

### Etapa 6: Pós-Venda

**Garantia de 30 Dias:**
- Cobre: Problemas na instalação, drivers faltando
- Não cobre: Vírus novos, danos físicos, problema de hardware

**Follow-up (3-7 dias depois):**
```
WhatsApp: "Oi [Nome]! Tudo certo com seu PC?
Se precisar de algo ou tiver dúvida, é só chamar.
Avalie nosso serviço: [link - quando tiver]"
```

**Solicitação de Indicação:**
```
"Se gostou do serviço, indica para amigos!
Ganhe R$ 10 de desconto na próxima formatação 
para cada indicação que fechar."
```

## 🛠️ Infraestrutura de Atendimento

### Ferramentas Utilizadas

**Hardware:**
- Notebook próprio (para trabalhar em campo)
- Pendrive bootável multissistema (Windows, Linux, ferramentas)
- Kit de chaves de precisão
- Cabos (SATA, USB, rede)
- Mouse/keyboard USB (para testes)
- HD externo (para backups)

**Software:**
- Rufus (criar pendrives bootáveis)
- Ventoy (multiboot)
- CrystalDiskInfo (diagnóstico de HD/SSD)
- CPU-Z (informações de hardware)
- Driver Booster (instalação de drivers)
- TeamViewer/AnyDesk (suporte remoto - ocasional)

**Ambiente:**
- Mesa limpa e organizada
- Iluminação adequada
- Internet estável
- Energia estável (nobreak recomendado)

## 📊 Eficiência Operacional

### Gargalos Atuais

**1. Agendamento Manual**
- Cliente liga/manda mensagem
- Técnico verifica disponibilidade mentalmente
- Pode haver conflitos de horário
- **Solução:** Calendário online integrado (já tem, mas subutilizado)

**2. Orçamento Manual**
- Cada orçamento é feito "na hora"
- Pode esquecer de incluir itens
- Tempo gasto em cálculos
- **Solução:** Tabela de preços padronizada, script de orçamento

**3. Deslocamento Domiciliar**
- Tempo perdido no trânsito
- Custo de combustível
- Improdutividade
- **Solução:** Incentivar atendimento balcão com desconto

**4. Estoque de Hardware**
- Não mantém estoque (compra sob demanda)
- Pode demorar para conseguir peças
- Perde vendas por não ter "na hora"
- **Solução:** Estoque mínimo de SSDs mais vendidos

**5. Follow-up Inconsistente**
- Nem sempre lembra de fazer follow-up
- Perde oportunidade de indicação
- **Solução:** Automação de lembretes

## ⚡ Otimizações Sugeridas

### Imediatas (custo zero)

**1. Forçar Uso do Agendamento Online**
- Responder WhatsApp com: "Agende aqui: [link]"
- Reduz troca de mensagens
- Cliente vê disponibilidade em tempo real

**2. Script de Orçamento**
- Criar template de mensagem com opções
- Copiar, colar, ajustar valores
- Padroniza comunicação

**3. Checklist Digital**
- App de notas ou planilha
- Marcar itens durante atendimento
- Evita esquecer etapas

**4. Incentivo ao Balcão**
- "Levando no balcão: 10% OFF"
- Reduz deslocamento
- Mais rápido para ambos

### Curto Prazo (baixo custo)

**5. Estoque Mínimo**
- Comprar 3-5 SSDs 240GB (mais vendido)
- Investimento: ~R$ 360-600
- Venda imediata, sem esperar fornecedor

**6. Maquininha de Cartão**
- Mercado Pago, SumUp, etc
- Taxa ~3%, mas aceita parcelamento
- Aumenta conversão

**7. Automação WhatsApp**
- Mensagens automáticas de confirmação
- Follow-up programado
- Lembretes de garantia
