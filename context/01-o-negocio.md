## 💼 Serviços e Preços

### 1. Formatação Completa

#### Windows 10
- **Preço:** R$ 25 (todas as versões: Home, Pro, Enterprise)
- **Inclui:**
  - Instalação do sistema
  - Drivers atualizados
  - Programas essenciais (Chrome, VLC, etc)
  - Ativação completa
- **Diferenciais:** Versão 22H2, build 19045, leve e estável

#### Windows 11
- **Preço:** R$ 100 (todas as versões)
- **Inclui:** Mesmo pacote do Windows 10
- **Requisitos:** Hardware compatível (TPM 2.0, Secure Boot)
- **Público:** PCs novos, usuários que querem interface moderna

#### Ghost Spectre (Versões Modificadas)
- **Preço:** R$ 50-100 (depende da versão)
- **Descrição:** Windows ultra-otimizado, sem bloatware
- **Vantagens:** +30% performance, mais leve, maior privacidade
- **Ideal para:** PCs antigos, gamers, profissionais de performance

#### Linux (Ubuntu, Arch, etc)
- **Preço:** GRATUITO (somente instalação)
- **Inclui:** Distribuição livre, drivers, software básico
- **Ideal para:** Usuários avançados, programadores, orçamento limitado
- **Opções:** Ubuntu (fácil), Arch (avançado), Mint, Pop!_OS

#### Dual Boot (Windows + Linux)
- **Preço:** R$ 50 (adicional)
- **Descrição:** Ambos os sistemas no mesmo PC
- **Vantagem:** Escolha na inicialização

#### Backup de Dados
- **Preço:** +R$ 20 (adicional na formatação)
- **Descrição:** Salvamento completo de arquivos antes da formatação
- **Mídia:** Cliente fornece HD externo ou pendrive

### 4. Otimização e Overclocking

#### Otimização de Sistema
- **Preço:** R$ 50
- **Inclui:**
  - Limpeza de arquivos temporários
  - Desativação de programas desnecessários
  - Configuração de inicialização
  - Otimização de SSD (TRIM, etc)
  - Ajustes de performance
- **Resultado:** +20-30% velocidade percebida

#### Overclocking Seguro
- **Preço:** R$ 50
- **Inclui:**
  - Overclock de CPU (dentro de limites seguros)
  - Overclock de GPU (se aplicável)
  - Otimização de RAM (XMP/DOCP)
  - Testes de estabilidade
  - Monitoramento térmico
- **Garantia:** Limites conservadores para segurança
- **Aviso:** Pode invalidar garantia do fabricante
